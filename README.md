# Talent Performance Pulse

Talent Performance Pulse is a backend system designed for an internal HR tool that assists managers in tracking and reviewing employee performance. It is built to store performance data, expose clean RESTful APIs, and support basic analytical queries for HR reporting.

## Basic Functionality

* **Employee Management**: Provides the ability to create and manage employee records, including their department, role, and joining date.
* **Performance Reviews**: Enables the submission of performance reviews for employees within specific review cycles, capturing numerical ratings (1–5) and reviewer notes.
